# 创意咖啡物语 修改器

基于内存修改的游戏修改器，适用于 Steam 版《创意咖啡物语》(Cafe Master Story) v1.32。

## 功能列表

| 热键 | 功能 | 说明 |
|------|------|------|
| F1 | 无限金钱 | 花钱时金额不减少 |
| F2 | 无限研究点 | 研究点数不减少 |
| F3 | 无限满意度 | 顾客满意度不减少 |
| F4 | 免费员工升级 | 员工升级费用为0 |
| F5 | 员工全属性999 | 服务/魅力/料理/感知全部999 |
| F6 | 饮品满属性 | 所有饮品参数为999 |
| F7 | 料理满属性 | 所有料理参数为999 |
| F8 | 套餐满属性 | 所有套餐参数为999 |
| F9 | 料理瞬间升级 | 料理升级所需经验为0 |
| F10 | 退出 | 恢复所有修改并退出 |

## 使用方法

### 环境要求

- Python 3.8+
- pymem (`pip install pymem`)

### 运行步骤

1. 启动游戏
2. 双击 `run_trainer.bat` 或运行 `python trainer_gui.py`
3. 点击"连接游戏"按钮
4. 使用 F1-F9 切换各项功能
5. F10 退出修改器（会自动恢复原始代码）

> **注意**: 可能需要以管理员身份运行才能正常读写游戏内存。

## 文件说明

- `trainer_gui.py` - GUI版本（推荐）
- `trainer.py` - 控制台版本
- `run_trainer.bat` - 启动脚本

## 技术原理

通过解析 IL2CPP 元数据定位关键游戏方法在 `GameAssembly.dll` 中的偏移地址，运行时使用 pymem 注入补丁字节码：

- **无限金钱**: NOP掉 `SubMoney` 中的减法指令
- **无限研究点/满意度**: 让对应减少函数直接返回
- **免费升级**: `GetLevelUpCost` 返回 0
- **属性999**: `GetParameter`/`GetPara` 返回 999
- **瞬间升级**: `GetNextExp` 返回 0
